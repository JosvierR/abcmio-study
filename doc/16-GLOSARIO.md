# 16. Glosario

## 16.1 Términos Técnicos

### A

**API (Application Programming Interface)**
Interfaz de programación de aplicaciones que permite la comunicación entre diferentes sistemas de software.

**Artisan**
Interfaz de línea de comandos incluida con Laravel para ejecutar tareas comunes.

**AWS S3 (Amazon Web Services Simple Storage Service)**
Servicio de almacenamiento en la nube de Amazon para guardar y recuperar datos.

**Autoload**
Carga automática de clases PHP mediante Composer sin necesidad de includes manuales.

### B

**Backup**
Copia de seguridad de datos para recuperación en caso de pérdida.

**Bcrypt**
Algoritmo de hashing de contraseñas utilizado por Laravel por defecto.

**Blade**
Motor de plantillas de Laravel para generar vistas HTML.

**Bootstrap**
Framework CSS para diseño responsive y componentes UI predefinidos.

### C

**Cache**
Almacenamiento temporal de datos para mejorar el rendimiento.

**Controller**
Componente MVC que maneja la lógica de negocio y responde a peticiones HTTP.

**CORS (Cross-Origin Resource Sharing)**
Mecanismo que permite recursos restringidos en una página web sean solicitados desde otro dominio.

**CRUD (Create, Read, Update, Delete)**
Operaciones básicas de persistencia de datos.

**CSRF (Cross-Site Request Forgery)**
Tipo de ataque malicioso que ejecuta acciones no autorizadas en nombre de un usuario autenticado.

**CSS (Cascading Style Sheets)**
Lenguaje de estilos utilizado para describir la presentación de documentos HTML.

### D

**DDEV**
Herramienta de entorno de desarrollo local basada en Docker.

**Dependency Injection**
Patrón de diseño donde las dependencias se inyectan en lugar de crearlas internamente.

**DTO (Data Transfer Object)**
Objeto que transporta datos entre procesos.

**Dropzone**
Librería JavaScript para subida de archivos drag-and-drop.

### E

**Eloquent**
ORM (Object-Relational Mapping) de Laravel para interactuar con bases de datos.

**Environment (.env)**
Archivo que contiene variables de configuración específicas del entorno.

**Event**
Acción que ocurre en la aplicación y puede tener listeners asociados.

**Exception**
Error o condición anormal que interrumpe el flujo normal de ejecución.

### F

**Factory**
Patrón para crear objetos de prueba con datos falsos.

**Facade**
Interfaz estática a clases disponibles en el contenedor de servicios de Laravel.

**Feature Test**
Prueba que verifica el comportamiento de características completas de la aplicación.

**Foreign Key**
Clave en una tabla que referencia la clave primaria de otra tabla.

### G

**Gate**
Sistema de autorización de Laravel para verificar permisos de usuario.

**Guard**
Driver de autenticación en Laravel.

**Guzzle**
Cliente HTTP para PHP utilizado para hacer peticiones HTTP.

### H

**Hash**
Función que convierte datos en una cadena de longitud fija.

**HTTPS (Hypertext Transfer Protocol Secure)**
Protocolo seguro para transferencia de hipertexto.

**HTTP Status Code**
Código numérico que indica el resultado de una petición HTTP.

### I

**IDE (Integrated Development Environment)**
Entorno de desarrollo integrado.

**Intervention Image**
Librería PHP para manipulación de imágenes.

**IoC (Inversion of Control)**
Patrón de diseño donde el control de flujo es invertido.

### J

**Job**
Tarea asíncrona que se ejecuta en cola.

**JSON (JavaScript Object Notation)**
Formato ligero de intercambio de datos.

**JWT (JSON Web Token)**
Estándar para crear tokens de acceso.

### L

**Laravel Mix**
Wrapper de Webpack para compilar assets frontend.

**Listener**
Clase que maneja un evento específico.

**Locale**
Configuración regional que determina idioma y formato.

**Log**
Registro de eventos y errores de la aplicación.

### M

**Middleware**
Capa que filtra peticiones HTTP.

**Migration**
Archivo que define cambios en la estructura de base de datos.

**Model**
Clase que representa una tabla de base de datos y su lógica.

**MVC (Model-View-Controller)**
Patrón arquitectónico que separa la aplicación en tres componentes.

### N

**Namespace**
Espacio de nombres para organizar clases PHP.

**Nginx**
Servidor web y proxy inverso.

**NPM (Node Package Manager)**
Gestor de paquetes para JavaScript.

### O

**Observer**
Patrón que permite suscribirse a eventos de un modelo.

**OPcache**
Cache de bytecode de PHP para mejorar rendimiento.

**ORM (Object-Relational Mapping)**
Técnica para convertir datos entre sistemas incompatibles.

### P

**Package**
Librería o bundle de código reutilizable.

**Pagination**
División de resultados en múltiples páginas.

**PayPal**
Plataforma de pagos en línea.

**PDO (PHP Data Objects)**
Interfaz para acceder a bases de datos en PHP.

**PHP-FPM (FastCGI Process Manager)**
Gestor de procesos FastCGI para PHP.

**Policy**
Clase que organiza lógica de autorización.

**PSR (PHP Standard Recommendation)**
Estándares para código PHP.

### Q

**Query Builder**
Constructor de consultas SQL de Laravel.

**Queue**
Cola de trabajos para ejecución asíncrona.

### R

**Rate Limiting**
Limitación de número de peticiones en un período.

**React**
Librería JavaScript para construir interfaces de usuario.

**reCAPTCHA**
Servicio de Google para verificar que el usuario es humano.

**Redis**
Sistema de almacenamiento en memoria para cache y colas.

**Repository**
Patrón que abstrae la capa de acceso a datos.

**Request**
Objeto que representa una petición HTTP.

**Response**
Objeto que representa una respuesta HTTP.

**RESTful**
Estilo arquitectónico para APIs basado en HTTP.

**Route**
Definición de URL y su acción asociada.

### S

**Sass/SCSS**
Preprocesador CSS que extiende las capacidades de CSS.

**Seeder**
Clase que inserta datos de prueba en la base de datos.

**Service**
Clase que encapsula lógica de negocio reutilizable.

**Service Provider**
Clase que registra servicios en el contenedor de Laravel.

**Session**
Datos persistentes del usuario entre peticiones.

**Slug**
URL amigable generada desde un título.

**SMTP (Simple Mail Transfer Protocol)**
Protocolo para envío de emails.

**Spatie**
Empresa que crea paquetes populares para Laravel.

**SQL (Structured Query Language)**
Lenguaje para gestionar bases de datos relacionales.

**SSH (Secure Shell)**
Protocolo para acceso remoto seguro.

**SSL/TLS (Secure Sockets Layer / Transport Layer Security)**
Protocolos criptográficos para comunicaciones seguras.

**Supervisor**
Sistema para controlar procesos en Unix.

### T

**Telescope**
Herramienta de debugging para Laravel.

**Test**
Código que verifica el comportamiento correcto de la aplicación.

**Trait**
Mecanismo de reutilización de código en PHP.

**Transaction**
Secuencia de operaciones de base de datos que se ejecutan como unidad.

### U

**Unit Test**
Prueba que verifica el comportamiento de una unidad de código.

**URL (Uniform Resource Locator)**
Dirección de un recurso en internet.

**UUID (Universally Unique Identifier)**
Identificador único universal.

### V

**Validation**
Proceso de verificar que los datos cumplan reglas específicas.

**View**
Plantilla que genera HTML para mostrar al usuario.

**Vue.js**
Framework JavaScript progresivo para interfaces de usuario.

### W

**Webhook**
Callback HTTP para comunicar eventos entre sistemas.

**Webpack**
Empaquetador de módulos JavaScript.

### X

**XSS (Cross-Site Scripting)**
Vulnerabilidad de seguridad que permite inyectar scripts maliciosos.

## 16.2 Términos del Dominio

### ABCmio

**Anuncio**
Publicación de una propiedad en la plataforma.

**Categoría**
Clasificación de propiedades (Apartamentos, Casas, Comercial, etc.).

**Ciudad**
Ubicación específica dentro de un país.

**Crédito**
Moneda virtual de la plataforma usada para publicar propiedades.

**Directorio**
Listado público de propiedades disponibles.

**Expiración**
Fecha límite de visibilidad de una propiedad publicada.

**Extensión**
Prolongar el período de publicación de una propiedad.

**Galería**
Colección de imágenes de una propiedad.

**Orden**
Registro de transacción de compra o uso de créditos.

**Paquete de Créditos**
Conjunto de créditos disponibles para compra.

**País**
Ubicación geográfica de alto nivel.

**Propiedad**
Inmueble o servicio publicado en la plataforma.

**Publicación**
Acción de hacer visible una propiedad en el directorio público.

**Reporte**
Denuncia sobre contenido inapropiado de una propiedad.

**Tipo de Propiedad**
- **Venta**: Propiedad en venta
- **Alquiler**: Propiedad para rentar
- **Servicio**: Servicio inmobiliario

**Transferencia**
Envío de créditos de un usuario a otro.

**Usuario**
Persona registrada en la plataforma.

**Verificación de Email**
Confirmación de dirección de correo electrónico.

**Visita**
Acceso a la página de detalle de una propiedad.

## 16.4 Comandos Frecuentes

### Artisan

```bash
php artisan migrate              # Ejecutar migraciones
php artisan db:seed             # Ejecutar seeders
php artisan make:controller     # Crear controlador
php artisan make:model          # Crear modelo
php artisan make:migration      # Crear migración
php artisan cache:clear         # Limpiar cache
php artisan config:cache        # Cachear configuración
php artisan route:list          # Listar rutas
php artisan tinker              # REPL de Laravel
```

### Composer

```bash
composer install                # Instalar dependencias
composer update                 # Actualizar dependencias
composer dump-autoload          # Regenerar autoload
composer require package/name   # Agregar paquete
```

### NPM

```bash
npm install                     # Instalar dependencias
npm run dev                     # Compilar assets (desarrollo)
npm run watch                   # Compilar y observar cambios
npm run production              # Compilar assets (producción)
```

### DDEV

```bash
ddev start                      # Iniciar proyecto
ddev stop                       # Detener proyecto
ddev ssh                        # Acceder al contenedor
ddev exec php artisan migrate   # Ejecutar comando dentro
ddev logs                       # Ver logs
```
